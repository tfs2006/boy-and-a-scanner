SDS100/SDS200 Waterfall Display (FREE Firmware Update)

Great news for existing SDS100 and SDS200 owners: Uniden added waterfall/spectrum display capability to both scanners via a free firmware update in December 2023!

  • Firmware Version: v1.23.01 Main / v1.03.02 Sub (December 21, 2023)
  • Cost: FREE ‚Äî update via Uniden Sentinel software
  • Features: Graphical spectrum display showing RF activity across a user-selectable band
  • Updates Since: v1.23.05 (Feb 2024) fixed IF polarity bugs; v1.03.06 (July 2024) improved AM audio

This is a major feature add that puts the SDS100/SDS200 on par with the SDS150 for waterfall display. The SDS150's waterfall is built-in from the start and may have additional optimizations.

üÜï What's New in the SDS150?

The SDS150 is Uniden's newest flagship handheld, building on the SDS100's proven True I/Q platform with several major upgrades:

  • Built-in GPS Receiver: No more external GPS pucks or messy cables. The SDS150 has GPS built right in for automatic location-based scanning.
  • Built-in Bluetooth: Connect wirelessly to your smartphone for remote control via the new U/AWARE app (iOS and Android).
  • Waterfall Display: See RF activity visually on screen‚Äîbuilt-in from the start (SDS100/SDS200 also have this via firmware update).
  • U/AWARE Smartphone App: Control your scanner, view activity, and manage settings from your phone.
  • Improved SD Card Access: The microSD slot is now on the side for easy access without removing the battery.

  • Same Rugged Build: Water-resistant to JIS4/IPX4 standards, just like the SDS100.

What All Three Have in Common

Despite their differences, all three scanners share Uniden's core technology:

  • True I/Q SDR Receiver: The software-defined radio front end that excels at decoding P25 signals in challenging simulcast environments. This is the #1 reason to choose any of these scanners.
  • Digital Trunking: APCO P25 Phase I and Phase II are included. All three support optional DMR, NXDN, and ProVoice upgrade keys (~$50-75 each).
  • Waterfall Display: All three now support spectrum/waterfall display. SDS150 has it built-in; SDS100/SDS200 got it via December 2023 firmware update.
  • RadioReference Database: All ship with the complete RadioReference database for the USA and Canada.
  • Recording: All record to microSD and support instant replay.
  • Close Call RF Capture: All can detect nearby transmissions automatically.
  • Sentinel Software: All use Uniden Sentinel for PC-based programming and database updates.

Frequently Asked Questions

What is the difference between SDS100 and SDS200?

The SDS100 ($699.99) is a portable handheld scanner with battery power and water resistance. The SDS200 ($749.99) is a desktop/mobile base station with a larger speaker, Ethernet connectivity, and line-out/discriminator outputs. Both use identical True I/Q SDR receivers and support the same digital modes.

What is the difference between SDS100 and SDS150?

The SDS150 ($949.99) is the newer model with built-in GPS, built-in Bluetooth, waterfall spectrum display, U/AWARE smartphone app support, and an included charging cradle. The SDS100 ($699.99) requires external GPS, has no Bluetooth, but now has waterfall display via firmware update. Both are handheld and water-resistant.

Which Uniden scanner should I buy in 2025?

Buy the SDS100 ($699.99) for best value as a portable scanner. Buy the SDS150 ($949.99) if you want built-in GPS and Bluetooth without cables. Buy the SDS200 ($749.99) for a home base station with loud audio and Ethernet remote control.

Do the SDS100 and SDS200 have waterfall display?

Yes. Both the SDS100 and SDS200 received waterfall/spectrum display capability via free firmware update v1.23.01 released in December 2023. The SDS150 has waterfall display built-in from the factory.

How much do DMR and NXDN upgrade keys cost?

DMR, NXDN, and ProVoice upgrade keys each cost approximately $50-75 and are purchased through my.uniden.com. The same upgrade keys work on SDS100, SDS150, and SDS200. Keys are tied to your scanner's serial number and are non-transferable.
