Manual Programming P25 Systems

Create P25 trunked or conventional systems on your SDS100 without a computer.

Introduction

The SDS100 can decode APCO P25 digital communications out of the box. While most users rely on Sentinel software to import systems from the RadioReference database, you can also build systems manually using the scanner’s front panel. This is useful when the database is incomplete or when you need to quickly add a new talkgroup or frequency while you’re on the go.

What You’ll Need

  • Your Uniden SDS100 scanner
  • System details: control‑channel frequencies for a trunked system, or individual channel frequencies for a one‑frequency system
  • Talkgroup IDs (TGIDs) or channel names
  • The NAC code for conventional P25 channels if known (optional)

Steps

  1. Create or select a Favorites List: Press Menu, scroll to Manage Favorites and choose Review/Edit System. Select an existing Favorites List or highlight New Favorites List and press E/yes to create one. Give it a descriptive name such as “Local P25”.
  2. Add a new system: With your Favorites List highlighted, choose New System. When prompted for the system type, rotate the scroll knob until you see P25 Trunk for trunked systems or P25 One Frequency for conventional channels. Press E/yes to select【2248†2262】. Enter a name for the system (e.g. “State Police”).
  3. Add a site (trunked only): For a P25 trunked system you’ll be prompted to create a site. Enter a site name and then input the control‑channel frequencies. The SDS100 will automatically determine voice channels when scanning.
  4. Add departments and talkgroups: After creating the system and site, choose Add Department. Name the department (perhaps by county or agency). Then select Add Channel and input each talkgroup ID along with an alpha tag. Repeat for each talkgroup you wish to monitor. For conventional P25 channels, enter the frequency and NAC if known.
  5. Save and scan: When you’ve entered all TGIDs or channels, press Menu repeatedly until you return to the main scan screen. Make sure the Favorites List is enabled. The scanner will now track the P25 system you created.

Next Steps

Manual programming gives you fine‑grained control over what you monitor. After creating your system you can adjust options such as priority, alert tones and service types. If you later import the same agency via Sentinel, you can compare the database programming against your custom list to fill any gaps.
