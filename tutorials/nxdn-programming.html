Manual Programming NXDN Systems

Add conventional and trunked NXDN systems to your SDS100 and monitor public‑safety and business users.

Introduction

NXDN is a digital radio protocol used by many public‑safety agencies and commercial operators. Like DMR, it requires an optional upgrade key on the SDS100【252690829170062†L0-L0】. After activating the NXDN option you can manually enter both conventional and trunked NXDN systems directly on the scanner.

What You’ll Need

  • An Uniden SDS100 with the NXDN upgrade installed
  • Frequencies used by the NXDN system (control channels for trunked systems, or voice channels for conventional)
  • RAN (Radio Access Number) values for each channel if required
  • Talkgroup IDs or channel names

Steps

  1. Create or select a Favorites List: From the main menu select Manage Favorites → Review/Edit System. Choose a list to add your NXDN system or create a new one.
  2. Add a new system: Choose New System and scroll until you find the NXDN system type. If your scanner has the upgrade, options such as NXDN Trunk and NXDN One Frequency will appear. Select the appropriate type and give the system a name.
  3. Enter sites and frequencies: For trunked NXDN systems, create a site and input all control‑channel frequencies. Some NXDN networks use numerical LCNs similar to DMR; consult local documentation. For conventional channels, simply enter the frequency and RAN code if necessary.
  4. Add departments and talkgroups: Create a department and add each talkgroup ID you wish to monitor, assigning descriptive tags. For conventional channels, each channel entry can include a label and RAN.
  5. Save and begin monitoring: Exit out of the menus and ensure your new Favorites List is enabled in scan mode. Your SDS100 will now follow the NXDN system.

Next Steps

Because NXDN systems can vary widely in configuration, it’s helpful to research the specific network you’re monitoring. Use online resources such as RadioReference to find RAN codes and talkgroup assignments. As with DMR, you can add new talkgroups on the fly or adjust priority and alert settings for channels you need to hear immediately.
