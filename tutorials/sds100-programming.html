Programming the Uniden SDS100

Set up and organize your digital scanner using the Sentinel software.

Introduction

The Uniden SDS100 is a powerful handheld scanner capable of decoding P25, DMR and more. In this guide we’ll walk through the basics of programming the SDS100 using Uniden’s Sentinel software.

What You’ll Need

  • The Uniden SDS100 scanner
  • A Windows computer with Internet access
  • The latest version of Uniden Sentinel software
  • A microSD card reader (if not programming directly via USB)

Steps

  1. Install Sentinel: Download and install the Sentinel software from the Uniden website.
  2. Update the Firmware: Connect your scanner via USB and select “Mass Storage” mode. In Sentinel, choose Update → Update Firmware to ensure your SDS100 has the latest features.
  3. Update the Database: Still in Sentinel, choose Update → Update Master HPDB to download the latest RadioReference database.
  4. Create a Favorites List: Go to File → New Favorites List and give it a name. This list will hold the systems and channels you want to monitor.
  5. Add Systems: Use the Database tab to browse or search for agencies in your area. Right-click a system and select Append To Favorites List to add it.
  6. Organize by Service Type: Once systems are in your list, you can enable or disable specific service types (police, fire, EMS, etc.) to tailor your scan.
  7. Write to Scanner: With your Favorites List complete, select Scanner → Write to Scanner to transfer the programming to your SDS100.

Next Steps

Once you’ve mastered the basics, explore Sentinel’s options like adding custom alert tones, creating location‑based scanning profiles and discovering nearby frequencies. Be sure to check out our other tutorials for more advanced topics.
