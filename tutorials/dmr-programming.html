Manual Programming DMR Systems

Program MotoTRBO and DMR one‑frequency systems on your SDS100 and optimize trunking with LCN Finder.

Introduction

DMR (Digital Mobile Radio) is a popular digital mode used by businesses, utilities and amateur radio. The SDS100 can monitor DMR systems, but you’ll need the optional DMR upgrade to enable this mode【252690829170062†L0-L0】. Once unlocked, you can manually enter MotoTRBO trunked systems or single‑channel DMR “conventional” systems using the scanner’s menus.

What You’ll Need

  • A Uniden SDS100 with the DMR upgrade installed
  • The system’s control‑channel frequencies and logical channel numbers (for trunked systems)
  • Talkgroup IDs, colour codes and time slots
  • The ability to run the LCN Finder on an active system (optional but recommended)

Steps

  1. Create or select a Favorites List: Press Menu, select Manage Favorites then Review/Edit System. Pick an existing list or create a new one.
  2. Add a new system: Choose New System and select the appropriate type. For trunked MotoTRBO systems scroll to MotoTRBO Trunk (an optional system type) and press E/yes; for a single DMR channel choose DMR One Frequency【2248†2262】. Give the system a name.
  3. Enter site and frequencies: For trunked systems you’ll be prompted to add a site. Enter the site name and input all control‑channel frequencies. You may need to assign logical channel numbers (LCNs) to each frequency; if you don’t know them, leave the numbers blank for now and use the LCN Finder later.
  4. Run LCN Finder (optional): After creating the system and enabling it in scan mode, go to Menu → Analyze → LCN Finder. The scanner will analyse the trunked system and populate the correct LCNs for MotoTRBO or DMR Tier III networks【3360…L17】. Save the results and return to programming.
  5. Add departments and talkgroups: Within the system select Add Department, name it, then choose Add Channel to enter each Talkgroup ID. For each TGID specify the colour code and time slot if known. For DMR One Frequency channels, enter the frequency, colour code and time slot.
  6. Save and scan: Exit the menus and begin scanning. The SDS100 will follow the DMR system using the LCNs and TGIDs you programmed.

Next Steps

DMR can be complex with different tiers and trunking formats. Use the LCN Finder whenever you’re unsure of channel ordering. You can also experiment with custom alerts and service types to prioritize important talkgroups. As new talkgroups appear, add them directly on the scanner so you never miss a conversation.
